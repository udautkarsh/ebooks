Data Object Description

This class includes the information of vSAN disk mapping for vSAN datastore and the vSAN direct storages like VMFS datastores backing by SCSI disks. The vSAN direct storages are the host local storages managing by vSAN though it doesn't contribute storage resource to vSAN datastore.
