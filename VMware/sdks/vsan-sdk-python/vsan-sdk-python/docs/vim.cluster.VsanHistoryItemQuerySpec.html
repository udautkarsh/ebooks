Data Object Description

Generic class to define the query spec for vSAN cluster historical information.

Properties

Name Type Description
cleanAll*xsd:boolean

True to clean all of historical health cache after query. Default is false if it's not specified.
clusters*ManagedObjectReference[]
to a ClusterComputeResource[]

The target cluster list. If not specified, it will return the historical health check result for all the clusters.
end*xsd:dateTime

The end time querying from the historical result. If not specified, it will return all the historical health check result from the specified start time.
start*xsd:dateTime

The start time querying from the historical result. If not specified, it will return all the historical health check result til the end time.
